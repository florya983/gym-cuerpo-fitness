# Gym Cuerpo Fitness 

Proyecto web realizado con HTML y CSS.

##  Descripción

Sitio web de un gimnasio donde los usuarios pueden:
- Ver clases disponibles
- Ver promociones
- Contactar al gimnasio
- Acceder a formulario de registro

## Tecnologías usadas
- HTML
- CSS
